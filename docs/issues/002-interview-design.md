# Issue: 求人別面接設計（質問テンプレ/自動深掘り/不足情報）と要約・根拠パック

## 背景
- 採用担当者が「使う理由」を短時間で体感できる体験が必要
- 求人ごとに面接設計を出せれば一次面接の工数削減を訴求しやすい

## ゴール
- 求人コンテキストに基づく質問テンプレを提示する
- 直近回答に対する深掘り質問を自動提案する
- 不足情報の指摘を可視化する
- 面接要約と根拠パックを自動生成する

## 非ゴール
- 面接評価の自動スコアリング
- ATS連携や外部連携

## 要件
- 求人選択で面接設計が生成される
- 質問テンプレは入力欄へ挿入できる
- 直近回答の深掘り候補が表示される
- 不足情報の指摘が表示される
- 要約と根拠パック（参照フラグメント）が表示される

## 受け入れ基準
- 求人選択なしではガイドが非表示
- 面接設計の質問テンプレが表示される
- 面接中の回答に対して深掘り候補が表示される
- 要約タブで根拠パックが確認できる

## テスト/確認
- 求人選択→ガイド生成→テンプレ挿入の動線確認
- 深掘り候補がチャット送信後に更新されること
- 要約/根拠パックが表示されること
